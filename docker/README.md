# Admin.NET 通用权限开发平台

Admin.NET 是一个基于 .NET 的通用权限开发平台，旨在为企业级应用提供全面的权限管理解决方案。该项目支持多租户架构，并集成了多种现代技术，如微服务、分布式系统、以及前后端分离架构。

## 功能特性

- **多租户支持**：允许为不同的客户或组织提供独立的数据隔离环境。
- **权限管理**：包括用户管理、角色管理、菜单权限、数据权限等。
- **日志记录**：操作日志、访问日志、异常日志等详细记录系统运行状态。
- **支付集成**：支持支付宝和微信支付接口。
- **任务调度**：内置作业调度功能，支持定时任务和分布式任务。
- **字典与枚举**：提供字典管理和枚举转字典的功能。
- **文件上传**：支持多种文件存储方式，包括OSS、SSH等。
- **API JSON**：提供统一的API JSON查询接口。
- **消息通知**：支持邮件、短信、即时通讯等多种消息发送方式。
- **动态插件**：可以动态加载插件，扩展性强。

## 技术栈

- **后端**：使用 .NET Core 构建，支持跨平台运行。
- **前端**：采用现代前端框架，如Vue.js，实现响应式布局。
- **数据库**：支持多种数据库，包括SQL Server、MySQL、PostgreSQL等。
- **缓存**：集成Redis作为缓存机制。
- **搜索**：支持ElasticSearch进行高效的数据检索。
- **安全**：使用JWT进行身份验证，支持OAuth2.0、LDAP等认证方式。
- **部署**：支持Docker容器化部署，便于快速上线。

## 安装指南

### 环境准备

- 安装 .NET SDK
- 数据库服务器（如MySQL、PostgreSQL）
- Redis 服务器
- RabbitMQ 或其他消息队列服务
- Elasticsearch（可选）

### 部署步骤

1. 克隆仓库到本地。
2. 修改配置文件以适应您的环境。
3. 使用命令 `dotnet restore` 恢复依赖包。
4. 使用命令 `dotnet build` 编译项目。
5. 使用命令 `dotnet run` 启动应用。

## 使用说明

### 用户权限管理

- 创建用户并分配角色。
- 为角色分配菜单权限和数据权限。
- 管理用户登录和登出。
- 支持第三方登录（如微信、Gitee）。

### 日志管理

- 查看操作日志、访问日志、异常日志。
- 支持日志导出和清理。

### 任务调度

- 添加、更新、删除作业任务。
- 配置触发器，设置任务执行规则。
- 监控任务执行情况。

### 字典与枚举

- 添加、更新、删除字典类型和字典值。
- 枚举类型自动转换为字典数据。

### 文件上传

- 支持多种文件上传方式，包括Base64上传。
- 支持文件预览和下载。

### API JSON 查询

- 提供统一的API JSON查询接口，简化前后端交互。

### 消息通知

- 发送邮件、短信、即时消息。
- 支持模板消息，方便消息定制。
